# Zenvyro Invoices

A modern, feature-rich mobile application built with **Flutter** designed for freelancers, developers, and small businesses to effortlessly create, manage, and track invoices, generate professional PDFs, and analyze business revenue.

---

## 🚀 Key Features

* **Interactive Dashboard & Status Tracking:**
* Real-time financial metrics including **Total Revenue**, **Total Invoices**, and breakdown of **Paid**, **Unpaid**, and **Overdue** items.
* Fast search functionality to find records by customer name or invoice ID instantly.
* Quick status filters to isolate specific payment states.


* **Dynamic Invoice Creation:**
* Itemized billing forms with custom quantities, unit prices, and line-item discounts.
* Automated subtotal, tax calculation, and grand total processing.
* Quick-select history and product catalog integration.


* **Professional PDF Generation:**
* Clean, print-ready document layouts featuring custom company logos, itemized tables, and tax breakdowns.
* Embedded **"Scan to Pay"** QR code integration and flexible payment instructions.


* **Revenue Insights & Analytics:**
* Visual bar charts and monthly earnings trend analysis to monitor business performance.


* **Comprehensive Settings & Data Management:**
* Company profile configuration (Business Name, Tagline, Address, NTN/Tax Number, Email, and Phone).
* Custom invoice prefixes, currency selection, and default tax rate settings.
* Full data portability with **CSV Export**, **Backup**, and **Restore** tools.


* **Dark & Light Mode:**
* Polished aesthetic with full support for a seamless dark theme experience.

## 🛠️ Tech Stack & Architecture

* **Framework:** Flutter / Dart
* **State Management & Services:** Provider / Custom Services
* **Database & Storage:** Local Database Helper (`sqflite`) / File Storage
* **PDF & Printing:** `pdf` & `printing` packages
